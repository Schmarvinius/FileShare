# FileShare

## Setup
Requirements:
- local copy of the repository
- node installation


### Backend
Starting in the root directory of the projects copy:

#### Initialization

1 - navigate into the backend folder
```zsh
cd backend/
```
2 - init
```zsh
npm i
```

#### Starting the Backend

1 - navigate into the backend folder
```zsh
cd backend/
```

2 - start the project
```zsh
node server.js
```

### Frontend

#### Initialization
1 - navigate into the backend folder
```zsh
cd frontend/
```
2 - init
```zsh
npm i
```

#### Starting the Frontend

1 - navigate into the backend folder
```zsh
cd frontend/
```

2 - start the project
```zsh
npm start
```
